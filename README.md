# ScalaHelloWorld
HelloWorldと表示

## 処理
画面に`Hello World`と表示

## コード
```
object Hello {
    def main(args: Array[String]): Unit = {
        println("Hello World")
    }
}
```

## 出力結果  
```
Hello World
```
